Data shaper

Compose a custom data shape — pick the axes (reviewer / reviewee / instrument / response field), the grouping, and the aggregations — and export the result alongside the canned lens CSVs.

{# Axis chip row. Two **mutually exclusive** axis chips — Reviewer / Reviewee — followed by the pipe, then the per-session **instrument scope chips** (always visible, independent of axis selection), then the slot the progressive-enhancement JS fills with the relevant column chips for whichever axis is currently on. Instrument scope chips are session-level filters — they apply to whichever axis is selected, not to a specific axis pool. With no instrument chip selected the aggregates span every session instrument, matching the legacy "By reviewer" / "By reviewee" general-data framing. Selecting one or more scopes the aggregates to those instruments. They use a distinct ``data-shaper-instrument-chip`` attribute (vs the per-axis ``data-shaper-col-chip`` vocabulary) so the progressive-enhancement JS can treat them as scope filters, not column-producing chips. #}
